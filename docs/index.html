Frequently Asked
Got questions? We've got answers.
Do I need a server or backend?
No. The app runs entirely in the browser using Firebase for cloud sync. You can host it on any static hosting (Firebase Hosting, Netlify, Vercel, or even a simple web server). No PHP, Node.js, or database server needed.
Does it work offline?
Yes. The app is a Progressive Web App (PWA) with a service worker. It caches the app shell and works offline. Data syncs to Firebase when you reconnect. You can install it on any device like a native app.
Is Firebase free?
Firebase has a generous free tier (Spark plan) that covers most small to medium shops. It includes 1GB database storage, 10GB/month transfer, and 50K reads/day. For larger operations, the Blaze plan charges per usage.
Can I customize the design?
Yes. The app uses CSS variables for theming. You can change colors, fonts, and layout. The invoice designer lets you customize receipt templates with your logo, colors, and footer text.
What devices does it support?
Any device with a modern browser — desktop, tablet, or phone. It's a PWA so it installs on Android and iOS. The mobile scanner companion app works on any phone with a camera.
Do you offer setup assistance?
Extended license includes Firebase setup assistance. We'll help you configure your Firebase project, set security rules, and get your first shop running.
Can I use it for multiple shops?
The Extended license includes multi-branch support. Each branch can have its own Firebase database or share one with branch-level filtering. Stock transfers between branches are supported.
What about data security?
All data is stored in your own Firebase project — we don't have access. Firebase provides enterprise-grade security with authentication, security rules, and encrypted connections. You control who can access what with role-based permissions.
